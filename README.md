<h1>🌍 U.S.–Iran Conflict: GCC Impact & Economic Analysis Dashboard</h1>

<p>
This project presents a <strong>Power BI dashboard</strong> analyzing the impact of the 
U.S.–Iran conflict on GCC (Gulf Cooperation Council) countries. 
The dataset focuses on incidents, infrastructure damage, economic loss, and risk distribution across the region.
</p>

<hr>

<h2>📁 Project Overview</h2>

<p>
The dashboard provides a comprehensive view of how geopolitical tensions affect regional stability and economies.
It highlights:
</p>

<ul>
  <li>Number and severity of conflict-related incidents</li>
  <li>Types of weapons used</li>
  <li>Damage to critical infrastructure</li>
  <li>Economic losses (GDP impact)</li>
  <li>Country-level risk analysis</li>
</ul>

<hr>

<h2>🛠 Tools Used</h2>

<ul>
  <li><strong>Power BI</strong> – Data modeling, visualization, and dashboard design</li>
  <li><strong>Data Cleaning & Transformation</strong> – Structured dataset for analysis</li>
</ul>
